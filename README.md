# Wisp — Releases & Update Feed

Public distribution + auto-update feed for **Wisp**, the iOS GPS location
simulator by **Business Lion LLC**. (The app's source code is private; this repo
hosts only the notarized, signed release binaries and the Sparkle appcast.)

- **Download:** see [Releases](../../releases).
- **Update feed:** [`appcast.xml`](appcast.xml) — the app checks this automatically.

Updates are Apple-notarized **and** signed with an EdDSA key, so only genuine
Business Lion LLC builds install.
